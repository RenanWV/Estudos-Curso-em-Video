como converter números para strings e vice e versa

                É bem simples siga o exemplo:
                    var n1 = Number(prompt('Digite um numero:'))
                    var n2 = Number(prompt('Digite outro numero:'))
                    var soma = n1 + n2
                    alert('A soma dos valores é: ' + soma.toString()).
                    você está convertendo uma string para números usando (Number) para fazer a conversão. É o (.toString()) para converter em string novamente

                    dica: use o template string para deixar seu código mais bonito ex:"alert(`A soma entre ${n1} e ${n2} é igual a ${soma}`)" usando o comando ${} e sempre utilizando a crase(``)

Formatação de números e strings

                Algumas formatações de strings:
                    var nome = prompt('Qual é seu nome?')
                    document.write(`Seu nome tem ${nome.length} letras.`)
                    document.write(`Seu nome em maiúscula fica ${nome.toUpperCase()}`)
                    document.write(`Seu nome em minúscula fica ${nome.toLowerCase()}`)
            

                Algumas formatações de números:
                    var n1 = 1545.5
                    n1.toFixed(2) = '1545.50' ele coloca mais casas decimas depois do (.)
                    n1.toFixed(2).replace('.', ',') = '1454,50' Ele troca um objeto por outro aqui no caso ponto(.) por vírgula(,).
                    n1.toLocaleString('pt-br', {style: 'currency', currency: 'BRL'}) = R$ 1,545.50 Ele formata para dinheiro REAL.

Operadores aritméticos

                5 + 2 = 7   (+ = soma)
                5 - 2 = 3   (- = subtração)
                5 * 2 = 10  (* = multiplicação)
                5 / 2 = 2.5 (/ = divisão)
                5 % 2 = 1   (% = resto de divisão inteira)
                5 ** 2 = 25 (** = potencia)
